This is a little add-on for ABC4EFT that helps to obtain the tree level UV dictionary of the operator types in the SMEFT.

One needs to load the ABC4EFT(https://github.com/XiaoMing-lei/ABC4EFT) first, then load this add-on "GetUVsForType.m".

It has only one function GetUVsForType, its usage can be found in the notebook example.nb.
